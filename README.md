# WISA: Iran's First Homegrown Open Source Project

*Featured in Linux Magazine — April 2004*

> "We tested more than 40 Linux distros, from Red Hat to Knoppix, and decided to use customized combinations of different distros on our servers. But the future plan is to build our own distro from scratch!"
>
> — Davoud Seyedin Boroujeni, Project Initiator

## About WISA

**WISA** (Web Integrated System Architecture) was a pioneering project deployed at the provincial unit of the Iranian Ministry of Industries and Mines in Kerman. It was the country's first homegrown IT system built entirely on Linux and Open Source software.

### Key Features

- Paperless automated office workspace
- Secure teleworking system via Internet
- E-forms for more than 30 separate applications
- Online decision workflow support
- Linux thin clients with KDE

### Technical Stack

- **Web Servers:** Apache
- **Database:** MySQL
- **Languages:** C++, PHP, Perl, Python, Ruby
- **OS:** Linux (customized combination of distributions)
- **Desktop:** KDE

## The Recognition

The project was featured in the "World News" section of **Linux Magazine** (Issue 41, April 2004) under the title:

> **"Iran Proudly Presents its First Open Source Project"**

This international recognition documented a significant milestone in the adoption of Open Source software in Iran.
